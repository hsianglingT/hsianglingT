Glossary

HTML
HTML stands for Hypertext Markup Language. It is designed for creating and structuring documents on websites.
CSS
CSS stands for Cascading Style Sheets. It is used to control the presentation and layout of HTML elements.
Hyperlinks
The <a> element is used to create hyperlinks, allowing users to navigate between different documents or web pages.
Unordered List
The <ul> element is used for listing items without a specific numerical order.
lang Attribute
The lang attribute defines the language of the content on a website.
Example: <html lang="en">...</html>
Here, en indicates that the content is in English.
